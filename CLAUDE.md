# FirstJob Widget — Project Memory

## Правила работы с кодом

### Консистентность правок
**Любое изменение UI или логики перед применением проверять на консистентность со всеми существующими сценариями:**
- `funnelState`: `applied` / `returning` (called=0, reject=0) / `returning` (есть ответы) / reviewed
- Экраны: filters (шаги 1–4), contract, day1 (воронка), day2 (статусы), prep (подготовка к звонкам)
- Кнопки и тексты: одно изменение может затрагивать несколько стейтов одновременно (updateFunnelFromStatuses управляет кнопками сразу для нескольких веток)
- Стили: изменение класса (btn-primary / btn-secondary / btn-link) в одном месте проверять во всех местах, где этот элемент используется

## Архитектура

### Основные состояния воронки (`funnelState`)
- `applied` — пользователь только что отправил отклики (до первого возврата на экран воронки)
- `returning` — пользователь вернулся на экран воронки:
  - если `called=0 && reject=0` → показываем таймер ожидания 24ч
  - если есть ответы → просим обновить статусы
- `reviewed` — статусы обновлены, показываем новую стратегию

### Ключевые функции
- `updateFunnelFromStatuses()` — обновляет счётчики воронки и кнопки по текущему `funnelState`
- `goToFunnel()` — переход на экран воронки, переключает `applied` → `returning`
- `updateFilterProgress()` — переключает шаги фильтра, двигает счётчик вакансий в `btn-stack` активного шага
- `startCountdown(seconds)` — запускает таймер обратного отсчёта
- `renderStatusCard()` — рендерит карточку вакансии на экране статусов
- `renderVacCard()` — рендерит карточку вакансии для отклика

### Стили кнопок
- `.btn-primary` — синяя, `font-weight: 400`
- `.btn-secondary` — серая, `font-weight: 500`
- `.btn-link` — текстовая ссылка, серая
- Чипы выбора (`.chip.selected`, `.schedule-chip.selected`, `.sphere-card.selected`) — светло-голубая заливка `rgba(39,155,240,0.16)` + синяя рамка

## История ключевых решений
- Счётчик вакансий перемещён из шапки фильтра в `btn-stack` активного шага (над кнопкой «Далее»)
- После прохождения подготовки к звонкам (первый раз, нет звонков) — показывать таймер 24ч, кнопки: «Подготовиться к звонкам» (primary) + «Уже позвонили?» (secondary)
- На экране статусов: сначала «Работодатель позвонил?» + кнопки выбора, затем «Что делать, когда позвонят» + аккордеон
